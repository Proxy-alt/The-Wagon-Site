A extension hanger exploit (works on 129)

  1. Download the files
  2. Insert the emoji file into the html file
  3. Click copy to clipboard
  4. Click on the extension you want to disable
  5. In a input/text field do control v and after a second do it again
  6. Open a page related to the extension and keep it open to continue the exploit

If you did it correctly every page related to it should just hang and not load forever.

Note: This only works on extensions that can be clicked on and have a place where you can input text. If you want to turn the extension back on just remove the tab that is trying to load or go to the extension page scroll down and switch "Allow access to file urls" on and then back off. If you don't succeed the first time try pasting it a different amount of times (like once or like 5 times), rereading the instructions or just tinker around with it until it works.
